 - [python](https://www.python.org/)
  - [flask](http://flask.pocoo.org/)
   - install

    ```
     pip install Flask
    ```
   - run

    ```
     python flask.py
   ```
  - [pyswip](https://pypi.python.org/pypi/pyswip)
   - install

    ```
     pip install pyswip
    ```
   -[flask-cors](https://flask-cors.readthedocs.io/en/latest/)
    - install
     ```
      pip install -U flask-cors
     ```
